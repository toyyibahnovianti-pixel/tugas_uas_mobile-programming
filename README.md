# BLUEPRINT PENGEMBANGAN APLIKASI - UAS MOBILE PROGRAMMING

## 1. Identitas Aplikasi
* **Nama Aplikasi:** WordRise (English Vocabulary Builder)
* **Judul Proyek:** Pengembangan Aplikasi Mobile Pembelajaran Kosakata Bahasa Inggris
* **Tema:** Edukasi / Self-Improvement
* **Platform:** Android (Flutter)

## 2. Deskripsi Proyek
WordRise adalah aplikasi mobile yang dirancang untuk membantu pengguna memperkaya kosakata bahasa Inggris secara terstruktur. Aplikasi ini mengelompokkan kata berdasarkan kategori tema, menyediakan definisi yang jelas, contoh penggunaan, serta fitur kuis untuk memperkuat daya ingat pengguna.

## 4. Fitur Utama (Scope)
1.  **Categorized Learning:** Pengelompokkan kosakata berdasarkan tema (Bisnis, Perjalanan, Akademik).
2.  **Word Detail:** Penjelasan lengkap mengenai jenis kata, arti, dan cara penggunaan dalam kalimat.
3.  **Interactive Quiz:** Fitur uji pemahaman untuk mengevaluasi hafalan kosakata.
4.  **Local Storage:** Fitur bookmark untuk menyimpan kata-kata penting secara offline.
5.  **Elegant UI/UX:** Desain minimalis menggunakan Google Fonts (Poppins) agar nyaman dipandang.

## 5. Acuan Desain (Layout)
* **Tema Warna:** Deep Blue & Soft Grey (Memberikan kesan profesional dan bersih).
* **Tipografi:** Menggunakan Google Fonts untuk memastikan keterbacaan yang tinggi.
* **Prinsip Layout:** Menghindari elemen yang terlalu ramai (cluttered) dan mengutamakan kemudahan navigasi bagi pengguna.

## 6. Dokumentasi & Pengujian
* Setiap fitur akan diuji melalui Emulator Android dan Real Device.
* Riwayat perubahan dan progres fitur akan selalu diperbarui pada file README.md di setiap akhir pekan.
